The Witcher 3: Wild Hunt

الوصف: لعبة تقمص أدوار في عالم مفتوح، تروي قصة الصياد Geralt of Rivia.

متطلبات التشغيل (الحد الأدنى):

  • نظام التشغيل: 64-bit Windows 7 أو أحدث
  • المعالج: Intel Core i5-2500K / AMD Phenom II X4 940
  • الذاكرة: 6 GB RAM
  • كرت الشاشة: NVIDIA GTX 660 / AMD Radeon HD 7870
  • المساحة التخزينية: 35 GB

Red Dead Redemption 2

متطلبات التشغيل (الحد الأدنى):

  • المعالج: Intel Core i5-2500K / AMD FX-6300
  • الذاكرة: 8 GB RAM
  • كرت الشاشة: Nvidia GTX 770 / AMD Radeon R9 280

Cyberpunk 2077

الوصف: لعبة عالم مفتوح تدور في مدينة Night City، تقدم قصة اللاعب "V" الذي يحاول الوصول للخلود الرقمي.

متطلبات التشغيل (الحد الأدنى):

  • المعالج: Intel Core i5-3570K / AMD FX-8310
  • الذاكرة: 8 GB RAM
  • كرت الشاشة: GTX 780 / Radeon RX 470
  • المساحة التخزينية: 70 GB
